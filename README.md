# Banno Interview Exercise

## Description
Interview exercise for Banno (Jack Henry) designed to provide an endpoint for getting a forecast on a lat/lon pair

## Usage
```bash
sbt run
```

## Testing

### Prerequisites
- scala and sbt installed

### How to run unit tests
   ```bash
   sbt test
   ```
